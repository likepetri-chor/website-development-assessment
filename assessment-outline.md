> [!note]+ Brief
'Produce a website to promote the activities of a non-profit organisation. The aim of the website is to create awareness of the organisation and to encourage an audience of specific 13-to-19-year-olds to join. You must select a non-profit organisation, such as a charity, campaign group, community group, museum.
>
The website must be responsive to mobile devices'
>
>- Three web pages
>- Information about the organisation—its aims, activities, etc.
>- Information about an upcoming event, including a map of the location
>- One member’s personal story
>- An e-mail link to contact the organisation
>- A form to join the organisation
>- An interactive image slider with captions
>- Resource section with filter functionality (resources could include text, infographics and video)
>- Positive representations of the organisation and its members
>- Accessibility features
